# ex-cog-dev

Development versions of Claude Code plugins from the external-cognition toolkit:

- **research-toolkit** — Self-reflexive dialectical investigation protocols, frame rotation, iterative verification, STONK agent
- **vasana-system** — Pattern recognition across unrelated contexts - observe, record, test, browse behavioral patterns
- **makers-toolkit** — Build-discipline in command-with-reason register. system-pilot is the intrinsic-motivation adaptation of the original Universal CLAUDE.md Protocol; intrinsic-prompt-design extracts the methodology for adapting other directive prompts. Pair with research-toolkit:text-deconstruction for verification.

These are *dev* versions: newer, more experimental, and more frequently updated than the stable plugins published at [bogheorghiu/ex-cog](https://github.com/bogheorghiu/ex-cog).

## Install as a marketplace

```
/plugin marketplace add bogheorghiu/ex-cog-dev
/plugin install research-toolkit@ex-cog-dev
/plugin install vasana-system@ex-cog-dev
/plugin install makers-toolkit@ex-cog-dev
```

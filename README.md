# ex-cog-dev

Development versions of two Claude Code plugins from the external-cognition toolkit:

- **research-toolkit** — self-reflexive dialectical investigation protocols, frame rotation, iterative verification, STONK agent
- **vasana-system** — pattern recognition across unrelated contexts

These are *dev* versions: newer, more experimental, and more frequently updated than the stable plugins published at [bogheorghiu/ex-cog](https://github.com/bogheorghiu/ex-cog).

## Install as a marketplace

```
/plugin marketplace add bogheorghiu/ex-cog-dev
/plugin install research-toolkit@ex-cog-dev
/plugin install vasana-system@ex-cog-dev
```

# ex-cog-dev

Development versions of Claude Code plugins from the external-cognition toolkit:

- **research-toolkit** — Relentless self-reflexive dialectical thinking that questions its own premises. Investigation protocols (DIP, STONK), epistemic depth (text-deconstruction, negative-dialectical-spiral), dialectic testing, frame rotation, iterative verification, financial analysis
- **vasana-system** — Vasana System - interaction choreographies for collaborative cognition
- **makers-toolkit** — Build-discipline skills in command-with-reason register: system-pilot (engineering discipline for building deterministic systems) and intrinsic-prompt-design (meta-methodology of prompt design)
- **security-toolkit** — Threat-detection and dangerous-action-blocking hooks: tiered prompt-injection detection with allowlist, dangerous git operation blocking (push to main, force push, reset --hard, etc.), and Desktop Commander safeguards

These are *dev* versions: newer, more experimental, and more frequently updated than the stable plugins published at [bogheorghiu/ex-cog](https://github.com/bogheorghiu/ex-cog).

## Install as a marketplace

```
/plugin marketplace add bogheorghiu/ex-cog-dev
/plugin install research-toolkit@ex-cog-dev
/plugin install vasana-system@ex-cog-dev
/plugin install makers-toolkit@ex-cog-dev
/plugin install security-toolkit@ex-cog-dev
```

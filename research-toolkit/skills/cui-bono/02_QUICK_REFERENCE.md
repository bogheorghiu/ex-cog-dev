# Cui Bono Quick Reference

## Before Any Investigation

### 1. Clarify Framework
- What weighs heaviest? (weapons / labor / environment / governance / geopolitical)
- Absolute dealbreakers or graduated assessment?
- Which actors count as "imperial"? (US/NATO? China? Russia? Regional powers? All?)
- **Apply symmetrically** - same questions to all actors

### 2. Acknowledge Positioning
- What's my standpoint?
- Whose interests might this serve?
- What can't I see from here?

---

## Core Structure: Claims → Contradictions → Resolution

### CLAIMS
Document self-presentation and consensus narrative.

### CONTRADICTIONS (four methods)
| Method | Question |
|--------|----------|
| **Direct** | What evidence contradicts claims? |
| **Deductive** | If claims true, what else must exist? Does it? |
| **Falsification** | What would disprove this? Search for it. |
| **Adversarial** | What do critics/workers/communities say? |

### RESOLUTION
What's actually true given both? (Not "who wins")

### SECOND ANTITHESIS (after resolution)
| Question | What It Catches |
|----------|-----------------|
| **Who bears costs?** | Externalized burdens (maintainers, communities) |
| **Who captures benefits?** | Surplus flowing elsewhere than claimed |
| **What's multiplied?** | Existing asymmetries, not new value |

## Available Lenses

| Lens | Core Question |
|------|---------------|
| **Weapons** | Military contracts, dual-use, revolving door? |
| **Labor** | Safety, unions, violations, forced labor exposure? |
| **Environmental** | Contamination, violations, community impact? |
| **Governance** | Ownership, revolving door, regulatory capture? |
| **Geopolitical** | Which powers served? Centre/periphery position? |
| **Supply Chain** | Upstream sourcing, downstream end-users? |
| **Financial Flows** | Beneficial ownership, tax structures, money movement? |
| **Information** | Content control, surveillance enablement? |

## Evidence Quality Rating

| Rating | Meaning |
|--------|---------|
| **Verified** | Primary docs, court records, audited filings |
| **Corroborated** | Multiple independent credible sources |
| **Credible Allegation** | Single reputable source with methodology |
| **Contested** | Conflicting sources or potential bias |
| **Unverified** | Single source, no methodology |
| **Speculation** | Don't present as evidence |

### Admiralty System
Rate **source** (A-F) separately from **information** (1-6):
- A source can be reliable (A) while delivering doubtful information (4)

## Key Techniques

### ACH (Analysis of Competing Hypotheses)
- List ALL hypotheses
- Rate each evidence item against EACH hypothesis
- Focus on DISCONFIRMATION
- Evidence consistent with all hypotheses = zero diagnostic value

### Contradiction Analysis (Chinese)
- Identify principal contradiction (decisive one)
- Identify principal aspect (which side's development determines outcome)
- Follow the money, not the marketing

### Bulletproofing (African)
- Data supports story, never reverse
- Adversarial cross-examination
- Legal review
- Partner verification

### Hypothesis-Based Inquiry (ARIJ)
- Story is hypothesis until verified
- Willingness to abandon when contradicted

### Language/Power Analysis
When terms borrow legitimacy from other domains:
- **"MFN" provisions**: Trade treaty language for platform lock-in
- **"Protocol/standard"**: Technical neutrality masking power concentration
- **"Consent"**: Legal term where power asymmetry negates meaningful choice
- Ask: What legitimacy does this terminology borrow? Who benefits from framing it this way?

## Recomposition Questions

After applying lenses:
1. What pattern emerges only from combination?
2. What contradictions across lenses?
3. How do domains interact?
4. What would different geopolitical position emphasize?
5. What would affected communities say?

1. Entity: [reason + evidence quality]
2. Entity: [reason + evidence quality]

**Unresolved Contradictions**: [where evidence conflicts]

If [priority A] highest: [conclusion + trade-off]

## Critical Supplementary Questions

| Tradition | Question |
|-----------|----------|
| **Indigenous** | Who owns this knowledge? Whose interests served? |
| **Liberation** | Where's the preferential option for the poor? |
| **Feminist** | What do marginalized workers know that management doesn't? |
| **Gramscian** | What appears "natural" that was actually constructed? |
| **Dependency** | Is value flowing toward or away from periphery? |

---

## When to Use Full vs. Simplified

**Full multi-lens**: 3+ entities, 4+ domains, high stakes, complex trade-offs

**Simplified**: Single entity/domain, quick answer needed, low stakes

Even simplified: still use Claims→Contradictions→Resolution + evidence quality rating

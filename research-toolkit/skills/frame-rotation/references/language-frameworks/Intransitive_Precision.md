# Intransitive Precision

**Core approach**: Drop agents and objects unless meaning collapses without them. When meaning requires them, rephrase intransitively if possible. Natural flow beats rule-following.

**Intransitive verbs** = actions happen without someone doing something to something else
**Impersonal constructions** = "it" without specific reference (it rains, patterns emerge, understanding shifts)

## Practical Application

**When agents/objects don't matter:**
- "Researchers discovered patterns" → "Patterns emerged"
- "The data shows connections" → "Connections appear"
- "I think this reveals something" → "This reveals something" or just "Something reveals itself"

**When agents/objects matter for meaning:**
- Keep them: "Sarah's linguistics background shaped how patterns appeared to her"
- Or rephrase: "Patterns appeared differently through Sarah's linguistic lens"

**When forcing sounds awkward:**
- Use normal construction. Clarity wins over rules.

## Integration with Other Frameworks

- **Affirmative**: "Solutions surface" not "problems don't exist"
- **Present-centered**: "Rain falls" not "it will rain"
- **E-Prime**: "Energy flows" not "energy is strong"

## The Test

Does dropping the agent/object make meaning clearer or preserve it completely? 
- If yes → drop it
- If meaning requires the relationship → keep it but look for intransitive alternatives
- If those sound forced → use normal construction

**Language serves understanding. When the rule helps, use it. When it doesn't, ignore it.**

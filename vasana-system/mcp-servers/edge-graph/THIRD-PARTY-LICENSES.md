## Multiverse School - Autonomous Development System

**Source:** [Multiverse School](https://multiverse.school) by [@lizTheDeveloper](https://github.com/lizTheDeveloper)

The multi-layered memory architecture, agent-centric design patterns, and autonomous development
methodology were developed as part of the Multiverse School curriculum for building AI agents
that learn and improve across sessions.

**Key concepts derived:**
- Memory layers (Recent, Episodic, Compost, Core)
- Agent identity and cross-session persistence
- Self-improving agent patterns

## Anthropic Knowledge Graph Memory Server

**Source:** https://github.com/anthropics/anthropic-quickstarts/tree/main/mcp-servers/servers/memory

**License:** Apache License 2.0

The entity query layer (get_entity, search_nodes, read_graph, open_nodes, delete_relations)
was inspired by Anthropic's Knowledge Graph Memory Server API design. The implementation
differs significantly (relation-primacy vs entity-storage), but the tool interface maintains
compatibility for migration purposes.

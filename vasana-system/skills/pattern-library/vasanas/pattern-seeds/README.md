# Pattern-Seeds

Meta-patterns observed across multiple Vasanas.

## What Are Pattern-Seeds?

While Vasanas capture single conversation interaction patterns, Pattern-Seeds capture the *patterns of how patterns form*.

| Layer | Question Answered |
|-------|-------------------|
| Vasana | "How did we solve this problem together?" |
| Pattern-Seed | "How do we keep discovering solutions together?" |

## When to Create a Pattern-Seed

After noticing 3+ Vasanas share a recurring formation dynamic:
- Same type of productive friction
- Same discovery mechanism
- Same transformation point

## Structure

```markdown
# [Pattern-Seed Name]

## Core Pattern
[The minimal algorithmic form capable of reproducing this pattern]

## Vasanas Exhibiting This Pattern

[What to look for that indicates this pattern is active]

[How to consciously invoke this meta-pattern]
```
